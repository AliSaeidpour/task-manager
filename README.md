# Task Manager App

A modern task management app built with **React**, **TypeScript**, **Redux Toolkit**, **Material UI**, and **@dnd-kit**.  
Smooth UX with professional Drag & Drop for task reordering.

---

## Features

- Add tasks using **Enter** key or Add button  
- Filter tasks: All / Active / Completed  
- Toggle task completion with a single click  
- Smooth and professional Drag & Drop  
- Clear Completed tasks  
- Responsive and mobile-friendly design  

---
